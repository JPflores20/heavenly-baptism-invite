

# Digital Baptism Invitation Page

A beautiful, heavenly-themed baptism invitation with a **Soft Sky Blue** primary color and **Silver** accents, featuring all the essential sections for a complete digital invitation experience.

---

## Design Theme: "Heavenly & Soft"

**Color Palette:**
- **Background**: Clean white/cream (#FAFAFA) with subtle gradient to pale sky blue
- **Primary**: Soft Sky Blue (#E0F7FA / #B2EBF2)
- **Accents**: Silver (#C0C0C0 / #A8A8A8) for borders, icons, and decorative elements
- **Text**: Soft charcoal for readability

**Visual Elements:**
- DecorativeClouds component with fluffy cloud SVG illustrations for corners and dividers
- Subtle dove and star icons sprinkled throughout
- Rounded, soft shapes for inputs and cards
- Smooth gradient backgrounds instead of paper textures

---

## Page Sections

### 1. Hero Section
- Baby's name prominently displayed with elegant typography
- "Holy Baptism" headline with a soft sky/cloud background
- Date and a welcoming message
- Smooth scroll indicator

### 2. Music Player
- Floating music control for a soft lullaby/instrumental
- Play/pause toggle with silver icon styling
- Positioned unobtrusively (corner or bottom)

### 3. Countdown Timer
- Days, Hours, Minutes, Seconds until the ceremony
- Styled with sky blue cards and silver text
- Animated number transitions

### 4. Location Cards
- **Church Venue**: Church icon with ceremony location details, Google Maps link
- **Reception Venue**: Cake/celebration icon with party venue details
- Both cards with rounded corners and subtle shadow

### 5. Weather Section
- Shows weather forecast for the baptism day
- Light-themed weather icons matching the pastel aesthetic
- Helpful for guests to plan their attire

### 6. Itinerary Timeline
- **Ceremony** (e.g., 11:00 AM)
- **Photo Session** (e.g., 12:30 PM)
- **Lunch/Celebration** (e.g., 1:00 PM)
- Vertical timeline with cloud-like connectors

### 7. Photo Gallery
- Grid of baby photos (placeholder images initially)
- Lightbox view for full-screen photos
- Soft border styling

### 8. Gifts & Godparents Section
- Godparents' names and special message
- Optional gift registry link or preferences
- Tender styling with dove accents

### 9. RSVP Form
- Guest name input
- Attendance confirmation (attending/not attending)
- Number of guests
- Dietary restrictions or message field
- Rounded, soft-styled form inputs
- Submit confirmation with animated feedback

---

## Animations & Interactions
- Smooth fade-in animations as sections scroll into view (Framer Motion)
- Gentle hover effects on cards and buttons
- Floating cloud animations in the background
- Countdown number flip animations

---

## Responsive Design
- Mobile-first approach
- Stacked layout on mobile, side-by-side on desktop
- Touch-friendly buttons and inputs
- Optimized for sharing via WhatsApp/social media
